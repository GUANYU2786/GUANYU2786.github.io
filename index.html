Step 1 建立github 專案

Start a Project

Fill the Repository

填入自己的username會使你的網址成為https://username.github.io

這裡的username 指的是你的Github 帳號

Step 2 在本地安裝 Node.js /npm/ n套件

安裝 Node.js

安裝 Hexo

在網站設定檔最下方的repo 加入部屬的目的地

Step 5 Hexo clean / Hexo generate / Hexo deploy

  • 清除先前的檔案hexo clean或是 hexo cl
  • 建立檔案 hexo generate 或是 hexo g
  • 部屬到Github Page hexo deploy 或是 hexo d

Step 6 換主題 以 NexT 為例

首先要有 git 工具

  1. 找到喜歡的主題後進入github 的公開相關程式碼
  2. git clone 檔案網址 到網誌下的themes裡開心資料夾儲存
  3. 到 Hexo 配置文件下 將 theme : 修改成新主題的資料夾名
