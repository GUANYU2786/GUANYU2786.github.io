Hexo 的 NexT 教學

更換頭像、新增自介(about)、標籤(tags)與分類(categories)

更換頭像

在themes/NexT/_config.yml裡找到頭像路徑

可以直接更改圖片或是更改路徑指到圖片位置

新增自我介紹( about )

  1. 進入 blog 下的 NexT設定檔開啟需要的功能
    找到 Menu 去掉註解來開啟想要的功能
  1. 新增 about 頁面

hexo new page about

blog/source/ 下就會多一個 about 的資料夾
3. 更改 about/index.md 新增想要的內容

新增標籤( tags )

  1. 新增 標籤頁面
    hexo new page tags
  2. 進入新增的 tags/index.md 修改

    在第三行增加 type: “tags”

新增分類( categories )

  1. 新增 分類頁面
    hexo new page categories
  2. 進入新增的 categories/index.md 修改

在第三行增加 type: “categories”
